Orange

A simple systems development language

View on GitHub Download .zip Download .tar.gz

What?

Orange is an imperative, statically typed systems programming language that aims to be productive and powerful without all the clunkiness of C++.

Status

Orange is still in the design phase, so check back often for updates.

Goals

  • Support OOP
  • Compile to a native binary
  • Target desktop and development board platforms like Arduino
  • Pointers and inline assembly
  • Smart pointers
  • Runtime optional

Examples

extern int32 printf(char* s, ...)

for (int i = 0; i < 10; i++) 
    printf("Hello, world %d!\n", i)
end  

def swap(id a, id b) 
    temp = *a 
    *a = *b 
    *b = temp
end

int a = 3
int b = 5 

swap(&a, &b)
